# Epics

Multi-spec initiatives live here. One file per epic:

```
docs/epics/YYYY-MM-DD-<slug>.md
```

Each spec in an epic still gets its own Superpowers folder under `docs/specs/`.

**How to write epics:** `ai/skills/writing-epics/SKILL.md` (after `orun init --ai`:
`.cursor/skills/shared/writing-epics/SKILL.md`).
